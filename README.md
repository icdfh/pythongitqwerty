Используемые технологии

HTML5 (семантическая разметка)

CSS3 (Flexbox / Grid)

Адаптивная верстка

Реализовано

Корректная структура документа

Стилизация компонентов интерфейса

Адаптация под разные размеры экрана

Чистая и понятная организация файлов

Цель проекта

Закрепить базовые навыки верстки и понимание взаимодействия HTML и CSS без использования JavaScript.
